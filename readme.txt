This plugin allows you to add interactive image gallery using shortcode.

== Description ==

This plugin makes it possible to add image gallery with interactive animation and fully control in back-end. You can add your image in gallery as
a custom post. In every post you have to add title, featured image and image thumbnail. In settings 
page you can set main frame color, image shadow color, pop-up modal color and slider navigation color.

= Features =

* Easy to integrate new image.
* Add image thumbnail.
* Upload any number of gallery image.
* Fully gallery control option in back-end Settings page.

== Installation ==

1. Upload the `mlmalstic-slideshow-gallery` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. Add shortcode `[slideshow-image-gallery]` to your page.
4. Add image to your Admin `Slides` Menu page.

== Frequently Asked Questions ==

= How to use the plugin? =

1. Click 'Add New' in the 'Slides' menu and fill out your page.
2. Upload image via the uploader at the 'Featured Image' option. 
3. Upload thumbnail via the uploader at the 'Gallery Icon' metabox. 
3. Set color of frame, image shadow, pop-up modal and slide controller.
4. Add shortcode 'slideshow-image-gallery' to your page.
